# homebrew-sshenc

Homebrew tap for [sshenc](https://github.com/jgowdy/sshenc) -- hardware-backed SSH key management using macOS Secure Enclave.

## Install

```bash
brew tap jgowdy/sshenc
brew install sshenc
```

## Upgrade

```bash
brew upgrade sshenc
```

## Uninstall

```bash
brew uninstall sshenc
brew untap jgowdy/sshenc
```
